Stock Portfolio Tracker is a simple Python application that calculates the total investment value of a user's stock portfolio. Users enter stock names and quantities, while the program uses a predefined dictionary of stock prices to compute the total investment. The application also saves the final result to a text file for future reference. This project demonstrates the use of dictionaries, loops, conditional statements, user input/output, basic arithmetic, and file handling in Python.

Key Features:

Accepts stock names and quantities from the user.
Uses a predefined dictionary to store stock prices.
Calculates the total investment value.
Handles invalid stock names.
Saves the portfolio summary to a .txt file.

Technologies Used:

Python
Dictionary
Loops
Conditional Statements
File Handling
Basic Arithmetic
